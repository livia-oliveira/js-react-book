# js-react-book

A practical book about JavaScript and React — from intermediate to professional.

## About
This repository contains all code examples and exercises from the book **Junior to Senior**, a hands-on guide covering advanced JavaScript concepts and React in depth.

## Structure
Each chapter has two folders:
- `examples/` — code examples from the chapter
- `exercises/` — practice exercises
